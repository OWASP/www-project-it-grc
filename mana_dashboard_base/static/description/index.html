Related Modules

Blocks

This dashboard has a lot of blocks, and you can use them to build your dashboard. also you can create a new type of block, and use it in your dashboard.

Styles

You can set the style of the block

Property

You can set the property of the block

Search

This dashboard support complex search like the odoo domain selector.

Multi Type Datasoruce

This dashboard support multi type datasource, you can use the odoo model, or the sql query, or the python code to get the data.

Custom Script

You can write the custom script to do some special things.

Block Template

You can create the block template, and use it in your dashboard. and you also can crate a type of block, and use it in your dashboard.

Qweb Template

You can use a qweb template to render the list.
