Restrict Menu Items from Specific Users

Version 16.0 Community Enterprise

Overview

This Module Allows you to Hide the Menus for Specific Users.

HIGHLIGHTS

Hide Menu for a Specific User
Show limited Menus to Specific Users
Will not hide Menus for Admin users
Easy to use, simple and effective.

SCREENSHOTS

Goto Settings > Users & Comapnies > Users.

Goto Hide Menu Tab.

Specify Menus which you want to Hide for that User.
