Iframes Backend

Users can add owns iFrames backend

If you use external tools like Google Analytics, elasticsearch & kibana or some interesting websites you might want to give your users a simple way to embed these sites in their personal Odoo dashboard.

  • Add some required info like Url,Name,.. .
  • Choose the length and width for your iframe .
  • Each user can only access to his dashboards .
  • Choose one or more dashboards as "Main Dashboord", and it can be accessibles via the menu "Main Dashboard".




    • To add an iFrame ,go to Users and add security group "User" or "Manager".

    Dashboard --> Iframe Dashboard --> iFrames and create a new entry

    Once created a new iFrame,it can be accessible via the smart Button "Dashboard".
